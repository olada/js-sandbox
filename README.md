# JS-SANDBOX
Playing around with Javascript stuff.


## How-To:
* Install node (e.g. version 2.15.8)
* `npm install gulp -g`
* `npm install webpack -g`
* `npm install webpack-dev-server -g`
* `git clone this repo`
* navigate to repo, then:
* `npm install`
* `gulp build-and-serve`
* Navigate to <http://localhost:1337>
